
# Project R5.05


## Run Locally

Clone the project

```bash
  git clone https://github.com/lucas6922/Projet-R5.05/
```

Install dependencies

```bash
  npm install
```

Create local database

```bash
  npm run db:push
```

Seed database

```bash
  npm run db:seed
```

Start the server

```bash
  npm run dev
```
